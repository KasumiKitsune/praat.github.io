PointProcess: To TextGrid...

从每个选定的 PointProcess（点过程） 对象创建一个空的 TextGrid。

在此操作中，PointProcess 对象唯一被使用的信息是其起始和结束时间。

设置

Tier names（层名称）
您想创建的层的名称列表，以空格分隔。
Point tiers（点层）
您想设为点层（point tiers）的层级名称列表；其余的层将为区间层（interval tiers）。如果您不想创建点层，可以将此设置留空。

示例

如果 Tier names 为 "a b c"，且 Point tiers 为 "b"，则生成的 TextGrid 对象将包含一个名为 "a" 的区间层，一个名为 "b" 的点层，以及另一个名为 "c" 的区间层。
