Intro

本教程是 Praat 的入门指南。Praat 是一款用于分析、合成、处理语音信号，并能为您撰写论文及报告绘制高质量插图的计算机软件。建议您通读本教程的所有章节。

您可以使用窗口顶部的“1 >”（下一页）和“< 1”（上一页）按钮按顺序阅读，也可以通过点击蓝色超链接直接跳转到所需的信息页。

 Intro 1. 如何获取声音: 录音, 从磁盘读取, 通过公式创建.
 Intro 2. 声音的操作与处理: 保存声音到磁盘, 查看与编辑.
 Intro 3. 声谱分析
    声谱图：查看声谱图, 设置声谱图参数, 查询声谱图信息, 打印声谱图, Spectrogram 对象.
    频谱切片：查看频谱切片, 设置频谱切片参数, Spectrum 对象.
 Intro 4. 音高分析
    音高曲线：查看音高曲线, 设置音高曲线参数, 查询音高曲线信息, 打印音高曲线, Pitch 对象.
 Intro 5. 共振峰分析
    共振峰曲线：查看共振峰曲线, 设置共振峰曲线参数, 查询共振峰曲线信息, Formant 对象.
 Intro 6. 音强分析
    音强曲线：查看音强曲线, 设置音强曲线参数, 查询音强曲线信息, Intensity 对象.
 Intro 7. 标注
 Intro 8. 声音操控：包括 音高、 时长、 音强、 共振峰的操控。

此外还有一些更具针对性的专题教程：

 语音学相关：
   • 嗓音/声源分析（基频微扰、振幅微扰、噪声）：Voice
   • 听辨实验：ExperimentMFC
   • 声音文件操作
   • 滤波器与滤波操作
   • 源-滤波器合成
   • 声道模拟/发音合成
 机器学习与认知建模：
   • 前馈神经网络模型
   • 优选论（OT）学习算法
 统计分析：
   • 主成分分析（PCA）
   • 多维尺度分析（MDS）
   • 判别分析
 通用与基础功能：
   • 脚本编写与批处理
   • 演示窗口交互界面
   • 图形输出与打印
