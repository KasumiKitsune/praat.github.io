Create TextGrid...

从头创建一个新的 TextGrid（标注网格）。

设置

Start time (s)（开始时间，秒）
起始时间，通常为 0 秒。
End time (s)（结束时间，秒）
以秒为单位的结束时间，通常是音频的时长。
Tier names（层名称）
您想创建的层的名称列表，以空格分隔。
Point tiers（点层）
您想设为点层（point tiers）的层级名称列表；其余的层将为区间层（interval tiers）。如果您不想创建点层，可以将此设置留空。
