Save as text file...

Save 菜单 中的命令之一。

可用性

您可以在选择一个或多个 objects 对象后选择此命令。

行为

对象窗口将询问您文件名。 在您点击 OK 之后，选中的对象将被写入磁盘上的文本文件中。

文件格式

如果您只选择了一个对象，例如 Pitch 类型的对象， 文件将以以下几行开始：

File type = "ooTextFile"
 Object class = "Pitch"

在此之后，音高（Pitch）数据将紧随其后。

如果您选中了多个对象，例如 “Pitch hallo” 和 “Polygon kromme”， 该文件将看起来像：
File type = "ooTextFile"
 Object class = "Collection"
 size = 2
 item []:
 item [1]:
 class = "Pitch"
 name = "hallo"
 (pitch data...)
 item [2]:
 class = "Polygon"
 name = "kromme"
 (polygon data...)

该文件可以使用 Read from file... 重新读取， 顺便说一句，重新读取并不需要上述示例中的冗长格式。 以下最简格式也将被正确读取：

"ooTextFile"
 "Collection" 2
 "Pitch" "hallo" (pitch data...)
 "Polygon" "kromme" (polygon data...)

因此，所有不是独立数字、也不包含在双引号或 < > 中的文本， 均被视为注释，在同一行感叹号（“!”）之后的任何文本也是如此。
