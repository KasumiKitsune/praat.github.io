Query submenu

在您选中几乎任何类型的对象时，显示在 对象窗口 右侧的子菜单。

查询命令可以为您提供关于对象的信息。

大多数查询命令以单词 Get（获取）或有时以 Count（计数）开头。 您可以在两个地方找到这些命令：在选中列表中的 对象 后通常会出现在右侧的 Query（查询）子菜单， 以及各个编辑器（editors，例如 音高菜单、共振峰菜单 或 声谱图菜单）菜单的查询部分。

行为

如果您点击查询命令，答案将写入信息窗口（Info window）。

脚本编写

在脚本中，您仍可以使用查询命令将信息写入信息窗口， 但您也可以使用任何查询 command 将信息直接存入变量 （参见 Scripting 6.3. 查询命令）。在这种情况下，数值将不会写入信息窗口。
