TextGrid: Count labels...

查询选定的 TextGrid 对象中指定的层包含多少个指定的标签。

设置

Tier number（层编号）
您想查询的层的编号（1, 2, 3...）。
Label text（标签文本）
您想计数的标签文本内容。

行为

层级 Tier number 中包含 Label text 标签的区间或点的数量 将被写入到 信息窗口 中。如果指定的层不存在，则数量为 0。

脚本编写

您可以使用此命令将数量赋值给脚本变量：

selectObject: "TextGrid hallo"
 number_of_a = Count labels: 1, "a"

在此情况下，结果数值将不会写入到信息窗口中。
