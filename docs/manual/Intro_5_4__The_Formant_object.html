Intro 5.4. The Formant object

在 SoundEditor 或 TextGridEditor 窗口中可见的共振峰曲线， 可以作为一个单独的 Formant（共振峰）对象复制到对象列表（List of Objects）中。为此， 请选择 Formant 菜单下的 Extract visible formant contour（提取可见共振峰曲线）。

获取独立 Formant 对象的另一种方法是在列表中选中一个 Sound 对象， 然后选择 Sound: To Formant (burg)...（Burg 算法转换共振峰...）（推荐方法）或者 共振峰与线性预测分析菜单 下的任何其他算法。

将共振峰曲线保存到磁盘

要将共振峰曲线数据保存到磁盘，请在列表中选择该 Formant 对象，然后执行 保存菜单 中的命令。

稍后，您可以使用 打开菜单 中的 Read from file...（从文件读取...）重新读入保存的共振峰文件。

绘制共振峰曲线

要将 Formant 对象绘制到图像窗口（Picture window），请选中它并选择 绘制菜单 中的命令。 在图像窗口中，您可以对其进行打印、保存为 EPS 文件，或复制到剪贴板中以便插入文字处理软件。
