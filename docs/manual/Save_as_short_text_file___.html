Save as short text file...

Save 菜单 中的命令之一。

可用性

您可以在选择一个或多个 objects 对象后选择此命令。

行为

对象窗口将向您询问文件名。 在您点击 OK 后，选中的对象将被写入磁盘上的文本文件中。

文件格式

该格式比 Save as text file... 中描述的格式短得多。 大多数注释都消失了，并且通常每行只有一项数据。

该文件可以使用通用的 Read from file... 重新读取。
