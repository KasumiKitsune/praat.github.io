Types of objects

Praat 包含以下类型的对象和编辑器（Editors）。 有关入门说明和教程，请参阅 Intro。

通用目的：

• Matrix: 采样后的双变量实值函数
• Polygon: 多边形
• PointProcess: 点过程（PointEditor）
• Sound: 采样后的连续声波信号（SoundEditor, SoundRecorder, 声音文件）
• LongSound: 基于磁盘长音频文件的数据流（LongSoundEditor）
• Strings: 字符串列表
• Distributions, PairDistribution: 分布与配对分布
• Table, TableOfReal: 数据表与实数表
• Permutation: 排列
• ParamCurve: 参数曲线

周期性/基频分析：

• 教程：
   • Intro 4. 音高分析
   • Intro 6. 音强分析
   • Voice (jitter, shimmer, noise)（嗓音分析：基频微扰、振幅微扰、噪声比例）
• Pitch: 生理基频、声学周期性或感知音高（PitchEditor）
• Harmonicity: 谐噪比（周期度）
• Intensity, IntensityTier: 音强曲线/强度层
• Electroglottogram: 声带电控图

声谱/频谱分析：

• 教程：
   • Intro 3. 声谱分析
   • Intro 5. 共振峰分析
• Spectrum: 复数等间隔频率谱（SpectrumEditor）
• Ltas: 长期平均频谱
• 时频分析：Spectrogram (声谱图), BarkSpectrogram (Bark声谱图), MelSpectrogram (Mel声谱图)
• Formant: 声学共振峰轨迹
• LPC: 线性预测编码系数随时间的变化
• Cepstrum, CC, LFCC, MFCC: 倒谱及相关系数
• Excitation: 基底膜兴奋模式
• Excitations: Excitation 对象的集合
• Cochleagram: 耳蜗兴奋模式随时间的变化

标注与切分（参见 Intro 7. 标注）：

• TextGrid (文本网格标注，TextGridEditor)

听辨实验：

• ExperimentMFC: 多项选择强制听辨实验

声音操控与重合成：

• 教程：
   • Intro 8.1. 操控音高
   • Intro 8.2. 操控时长
   • Intro 8.3. 操控音强
   • 滤波处理
   • 源-滤波器合成
• PitchTier: 音高层（PitchTierEditor）
• Manipulation (操控对象，ManipulationEditor): 基于 重叠相加法
• DurationTier: 时长层
• FormantGrid: 共振峰网格

发音合成（参见 发音合成 教程）：

• Speaker: 模拟女性、男性或儿童的声道与发音特征
• Articulation: 发音规格快照（肌肉活动）
• Artword: 发音目标随时间的变化
• (VocalTract: 声道面积函数)

神经网络包：

• FFNet: 前馈神经网络
• PatternList: 模式列表
• Categories: 分类类别（CategoriesEditor）

数值与统计分析：

• Eigen: 特征向量与特征值
• Polynomial, Roots, ChebyshevSeries, LegendreSeries, ISpline, MSpline: 多项式与各类曲线/样条
• Covariance: 协方差矩阵
• Confusion: 混淆矩阵
• 判别分析: Discriminant
• 主成分分析: PCA
• Correlation, ClassificationTable, SSCP
• DTW: 动态时间规整

多维尺度分析：

• Configuration (配置，包括 Salience)
• Kruskal 分析: Dissimilarity (Weight), Similarity
• INDSCAL 分析: Distance, ScalarProduct
• 对应分析: ContingencyTable

优选论学习（参见 OT 学习 教程）

• OTGrammar (OT语法，OTGrammarEditor）

辅助事务

• WordList (词表), SpellingChecker (拼写检查器)
