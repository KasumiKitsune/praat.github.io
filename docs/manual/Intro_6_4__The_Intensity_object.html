Intro 6.4. The Intensity object

要打印音强曲线，或者将其保存为 EPS 文件或复制到剪贴板中以便插入到您的文字处理器中， 您必须首先在对象列表（List of Objects）中创建一个 Intensity（音强）对象。 您可以通过在 SoundEditor 或 TextGridEditor 窗口的 Intensity 菜单中选择 Extract visible intensity contour（提取可见音强曲线）， 或者在列表中选中一个 Sound 对象并选择 Sound: To Intensity...（转换为音强...）来完成此操作。 在上述任一情况下，列表中都会出现一个全新的 Intensity 对象。 要将该 Intensity 对象绘制到图像窗口（Picture window），请选中它并选择 Draw...（绘制...）命令。 从图像窗口中，您可以将其打印、保存为 EPS 文件，或直接复制到剪贴板。
