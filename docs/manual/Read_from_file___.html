Read from file...

Open 菜单 中的命令之一。

目的

从磁盘文件中读取一个或多个 objects 数据对象。

行为

可以识别多种类型的文件：

1. 结构如 Save as text file... 中所述的文本文件； 这些文件可以包含任何类型的对象，或者多个对象的集合。
2. 使用 Save as binary file... 生成的二进制文件（任何类型）。
3. LISP 文本格式的文件（仅适用于可以写入 LISP 文件的对象类型）。
4. 能够被 Praat 内部库所识别的文件。 例如，语音学库增加了对许多类型声音文件的识别器。

如果文件包含多个对象，这些对象将出现在列表中， 并且它们的名称将与使用 Save as text file... 或 Save as binary file... 保存它们时的名称完全一致。

示例

• 如果文件仅包含一个 Pitch 对象并且文件名为 "hallo.pit"， 则对象列表中会出现一个名为 "Pitch hallo" 的对象。 列表中可以存在多个具有相同名称的对象。
• 如果文件包含一个名为 "hallo" 的 Pitch 类型对象， 以及一个名为 "kromme" 的 Polygon 类型对象， 则对象列表中会出现两个对象， 分别称为 "Pitch hallo" 和 "Polygon kromme".
