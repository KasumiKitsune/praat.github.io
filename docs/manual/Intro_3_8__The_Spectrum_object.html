Intro 3.8. The Spectrum object

要计算整个声音的傅里叶频谱， 请选中一个 Sound 对象，并选择 Spectrum 菜单下的 To Spectrum...。 一个新的 Spectrum（频谱）对象将会出现在对象列表中。 若要查看、修改或聆听它的某些部分，请点击 View & Edit。 要打印它，请先选中它并执行相应的 Draw（绘制）命令，将该 Spectrum 对象绘制到图像窗口（Picture window）中。
