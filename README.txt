Praat 简体中文本地化版本（非官方版，基于官方源码构建）。

【使用说明】
双击运行 Praat.exe 即可开始使用。

【本地化版特性说明】
1. Windows 平台拖拽文件打开支持
   - 支持将音频（.wav）、标注（.TextGrid）或脚本（.praat）等文件从文件夹拖拽入窗口直接打开，支持同时打开多个文件。
2. 默认不主动打开图像窗口 (Praat Picture)
   - 启动时默认只打开对象列表窗口，以减少屏幕占用。
   - 如需使用图像窗口，可通过主窗口左上角的 [Praat] -> [打开图像窗口 (Praat Picture)] 重新打开。
3. 中英术语【对照表】按钮
   - 在主窗口（Objects）下方添加了【对照表】按钮。
   - 点击可查看整理的语音学与 Praat 术语中英对照表，方便在使用中核对英文原词。
4. 帮助手册汉化与微调优化
   - 手册内容汉化：对内置帮助手册的部分核心页面进行了简体中文翻译。
   - 滚动条优化：调整了滚动条高度计算逻辑，使其根据正文内容高度自适应，支持鼠标和键盘整页滚动。
   - 修复了底部作者签名字符可能被截断的问题，并消除了调整窗口大小时偶尔弹出的越界警告框。
