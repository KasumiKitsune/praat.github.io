Praat 简体中文本地化版本（非官方版，基于官方源码构建）。

【使用说明】
双击运行 Praat.exe 即可开始使用。

【本地化专属新增特性】
1. Windows 原生文件拖拽打开支持 (Drag-and-Drop)
   - 可直接将音频（.wav）、标注（.TextGrid）或脚本（.praat）等支持的文件从电脑文件夹拖拽入 Praat 的任意窗口，即可自动完成加载。支持一次性拖入并同时加载多个文件。
2. 启动时默认隐藏图像窗口 (Praat Picture)
   - 启动时仅默认打开干净的对象列表窗口，不再默认强制弹出图像窗口以节省您的屏幕空间。
   - 打开入口：若需使用图像窗口，请点击 Objects 窗口左上角的 [Praat] 菜单 -> 选择 [打开图像窗口 (Praat Picture)] 即可立即调出使用。
3. 专属中英术语 [对照表] 按钮
   - 在 Objects 主窗口底部的固定按钮区中，特别添加了【对照表】(Bilingual) 按钮。
   - 点击该按钮即可弹出一个详尽的、按“学科类别”与“首字母拼音”双向索引的专业语音学与 Praat 专属中英术语对照参考表。方便随时查阅，避免翻译歧义。
4. 汉化帮助手册的体验深度优化
   - 手册汉化：对 Praat 核心帮助手册本身（Manual 手册页）进行了全方位的精细简体中文翻译。
   - 滚动优化：彻底重构了原版滚动范围死板且过长的问题。现在滚动滑块尺寸和滚动刻度完全根据当前页面实际内容高度动态计算，支持高精度的拖拽及鼠标整页滚动翻页。
   - 显示与稳定性修复：修正了原版由于页面底边距不足导致的作者签名截断问题，并消除了全屏或任意缩放窗口时容易疯狂弹出的滚动条数值越界警告框，运行极度稳定。
